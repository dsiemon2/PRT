## Engagement Playbook

### Responding to Mentions
```markdown
## Positive Mention
- Like immediately
- Reply within 2-4 hours
- Be genuine, not corporate
- Use their pet's name if shared

Example:
Them: "Just got my order from @MPS and my cat is in love!"
Us: "This made our whole day! Give [cat's name] some scritches from us"

## Complaint/Issue
- Reply within 1 hour
- Acknowledge, apologize, move to DM
- Never get defensive publicly

Example:
Them: "@MPS my order is 5 days late, no updates??"
Us: "So sorry about this! DMing you now to get this sorted."

## Question
- Reply within 2 hours
- Be helpful and direct
- Link to resources if needed

Example:
Them: "@PRT do these boots run true to size?"
Us: "They run slightly narrow. If you're between sizes, we'd recommend sizing up. Our size guide helps: [link]"
```

## Customer Service Protocol

### Response Time Targets
| Type | Target Response |
|------|-----------------|
| Complaint | < 1 hour |
| Question | < 2 hours |
| Positive mention | < 4 hours |
| General engagement | Same day |

### Escalation Path
```markdown
## Level 1: Social Team
- General questions
- Product inquiries
- Simple issues

## Level 2: Customer Support
- Order issues
- Refund requests
- Complex problems
â†’ "Let me connect you with our support team"
â†’ Move to DM or email

## Level 3: Management
- Public PR issues
- Angry viral threads
- Legal concerns
â†’ Flag immediately
â†’ Don't engage until guidance
```

### Crisis Protocol
```markdown
## If Negative Situation Goes Viral

1. DO NOT delete tweets (screenshots exist)
2. Pause all scheduled content
3. Flag to management immediately
4. Draft response for approval
5. Respond only when approved
6. Monitor sentiment

## Response Framework
- Acknowledge the issue
- Take responsibility if warranted
- Explain what you're doing about it
- Provide timeline if possible
```
